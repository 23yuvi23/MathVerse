# MathVerse

## Overview

The **MathVerse** is a frontend-based web application that allows users to find and book tutors based on their class, board, and subject preferences. The platform provides an easy-to-use interface to filter tutors and book a session according to the userâ€™s needs.

## Features

- **Tutor Filtering**: Users can filter tutors based on class, board, and subject.
    
- **Book a Session**: Users can book a session with a tutor directly.
    
- **Contact Us Form**: A form is available at the end of the page that redirects users to their email client for direct communication.
    
- **About Us Section**: Provides details about the platform and its mission.
    
- **Services Section**: Lists the services offered by the platform.
    
- **Pricing Section**: Displays pricing details for different tutoring options.
    

## Technologies Used

- **HTML**: Structure of the web pages.
    
- **CSS**: Styling for the platform.
    
- **Bootstrap**: Used for responsive and pre-designed components.
    
- **Tailwind CSS**: Used for utility-first styling.
    
- **JavaScript**: Used for interactivity and filtering functionality.
    

## How to Use

1. Open the website in a browser.
    
2. Use the filtering options to find a tutor based on class, board, and subject.
    
3. Click on **Book a Session** to proceed with booking.
    
4. Fill in the **Contact Us** form to get in touch via email.
    

## Contact

For any queries, users can fill out the **Contact Us** form, which will redirect them to their email client to send an email.

## Future Enhancements

- **Backend Integration**: Adding a database to store tutor details and bookings.
    
- **User Authentication**: Enabling login and profile management.
    
- **Payment Gateway**: Integrating a payment system for booking tutors.
    
- **Real-time Chat**: Allowing direct communication between tutors and students.
